Notice :

Dézipper, bien conserver tous les fichiers ensemble.

Le fichier "test.html" permet la passation des 4 tests. Il est possible de sauvegarder et de recharger ultérieurement, que le test soit fini ou non. Dans la page de passation, on peut soit afficher les questions une à une (par défaut), soit cliquer sur "vue d'ensemble" et voir toutes les questions.

Ensuite, on clique sur "voir les résultats". Afin de faire passer le test directement sur une tablette ou un écran sans que le patient ait accès aux résultats pour modifier ses réponses, il y a un mot de passe qui est simplement "results". Dans tous les cas, c'est possible d'ouvrir directement le fichier "result.html" qui n'a pas de mot de passe.

Dans la page résultat, il y a les normes que l'IA a trouvées dans des papiers sur internet qui sont sourcés, c'est possible de les modifier. Il y a comme dans le matériel source le bouton pour changer le mode de calcul fort = faible ou fort = 2xfaible. C'est possible d'afficher les points que rapporte chaque question. Quand on clique sur "imprimer", ça affiche la fenêtre d'impression. La page intègre une disposition spécifique optimisée pour l'impression.

Navigateur conseillé pour une expérience optimale : Google Chrome, Chromium ou Vivaldi.
